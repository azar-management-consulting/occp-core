# Comece em 60 segundos

O OCCP é o control plane open source para agentes de IA autônomos. Cada ação
que seu agente executa é **planejada, filtrada por política, executada,
validada e registrada em auditoria** — tudo antes que o agente confirme
qualquer operação irreversível.

## Instalação

```bash
pip install occp
# or
npm install @occp/sdk
```

## Sua primeira execução verificada

```python
from occp import ControlPlane

cp = ControlPlane(policy="policies/finance.rego")

result = cp.run(
    agent="claude-opus-4-7",
    task="Refund order #4521 if return window is open",
)

# PLAN → VERIFY → APPROVE → EXECUTE → AUDIT
print(result.audit_chain_id)
```
