# Induljon el 60 másodperc alatt

Az OCCP nyílt forráskódú vezérlőréteg autonóm AI ügynökökhöz. Minden művelet,
amelyet az ügynök végrehajt, **megtervezett, házirend-ellenőrzött, végrehajtott,
validált és auditnaplózott** — még mielőtt az ügynök bármilyen visszafordíthatatlan
változtatást véglegesítene.

## Telepítés

```bash
pip install occp
# or
npm install @occp/sdk
```

## Az első igazolt futtatás

```python
from occp import ControlPlane

cp = ControlPlane(policy="policies/finance.rego")

result = cp.run(
    agent="claude-opus-4-7",
    task="Refund order #4521 if return window is open",
)

# PLAN → VERIFY → APPROVE → EXECUTE → AUDIT
print(result.audit_chain_id)
```
