# Your first agent

This guide assumes you've finished the [Quickstart](/docs/quickstart) and
have a working API key.

## 1 · Scaffold the project

```bash
npx create-occp-app@latest my-agent --template hello-agent --ts
cd my-agent
npm install
```

Or, if you prefer a manual setup, clone the reference template:

```bash
git clone https://github.com/azar-management-consulting/occp-core
cp -r occp-core/templates/hello-agent my-agent
cd my-agent
```

## 2 · Wire credentials

```bash
cp .env.example .env
# edit .env → OCCP_API_KEY=occp_live_sk_...
```

## 3 · Run it

```bash
npm start
```

Expected output:

```
[hello-agent] sending task to https://api.occp.ai ...
[hello-agent] pipeline result: pass
[hello-agent] output: Hello from your first OCCP agent ✓
```

## 4 · Add a tool

Drop a file under `src/tools/` using the `echo.js` shape:

```js
export const myTool = {
  name: "search_docs",
  description: "Search OCCP documentation",
  parameters: {
    type: "object",
    properties: { query: { type: "string" } },
    required: ["query"],
  },
  async run({ query }) {
    const response = await fetch(
      `https://docs.occp.ai/api/search?q=${encodeURIComponent(query)}`,
    );
    return response.json();
  },
};
```

## 5 · Deploy

Push the repo to GitHub and click **Deploy to OCCP** from the dashboard.
The workflow in `.github/workflows/deploy.yml` handles the rest.
