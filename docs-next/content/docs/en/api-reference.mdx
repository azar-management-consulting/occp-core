# API Reference

The OCCP REST API v1 exposes **50+ endpoints** covering the full Verified
Autonomy Pipeline — planning, policy evaluation, execution, audit
retrieval, kill-switch control, and tenant/billing management.

<Callout type="info">
  A first-class, searchable, try-it-now reference is available at
  [/api-reference](/api-reference) (powered by Scalar). The raw OpenAPI
  spec lives at [`https://api.occp.ai/openapi.json`](https://api.occp.ai/openapi.json).
</Callout>

## Base URL

```
https://api.occp.ai/api/v1
```

All requests require `Authorization: Bearer occp_live_sk_...` unless
explicitly marked public (e.g. `/status`, `/openapi.json`).

## Quick ping

```bash
curl -sS https://api.occp.ai/api/v1/status | jq
```

Expected response:

```json
{
  "status": "ok",
  "version": "0.10.1",
  "region": "eu-central",
  "pipeline": "ready"
}
```

## Endpoint groups

| Group        | Prefix               | Highlights                                   |
| ------------ | -------------------- | -------------------------------------------- |
| Pipeline     | `/runs`              | `POST /runs`, `GET /runs/{id}`, stream SSE   |
| Plans        | `/plans`             | Draft, verify, diff                          |
| Policy       | `/policies`          | CRUD, evaluate, bundle                       |
| Audit        | `/audit`             | Chain queries, Merkle proofs                 |
| Kill switch  | `/admin/kill-switch` | Arm / disarm (Art. 14)                       |
| Tenants      | `/tenants`           | Org, members, quotas                         |
| Billing      | `/billing`           | Invoices, usage, limits                      |
