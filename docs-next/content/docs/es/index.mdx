# Comience en 60 segundos

OCCP es el plano de control de código abierto para agentes de IA autónomos.
Cada acción que ejecuta su agente es **planificada, controlada por políticas,
ejecutada, validada y registrada en auditoría** — todo antes de que el agente
confirme cualquier cambio irreversible.

## Instalación

```bash
pip install occp
# or
npm install @occp/sdk
```

## Su primera ejecución verificada

```python
from occp import ControlPlane

cp = ControlPlane(policy="policies/finance.rego")

result = cp.run(
    agent="claude-opus-4-7",
    task="Refund order #4521 if return window is open",
)

# PLAN → VERIFY → APPROVE → EXECUTE → AUDIT
print(result.audit_chain_id)
```
