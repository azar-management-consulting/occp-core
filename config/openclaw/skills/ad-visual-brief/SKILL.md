## Brief Structure

Produce a complete creative brief that a designer can execute without verbal clarification.

## Required Brief Sections

**1. Campaign Context**
- Product/service in one sentence
- Target audience: demographics + psychographics
- Primary message: one sentence (the single thing the viewer must remember)
- Desired emotion: (confidence / urgency / trust / curiosity / aspiration)

**2. Format Specifications**
| Format | Dimensions | File Type | Max Size |
|--------|-----------|-----------|----------|
| FB Feed | 1080×1080 | JPG/PNG | 1MB |
| FB Story | 1080×1920 | JPG/PNG/MP4 | 4MB |
| IG Feed | 1080×1080 | JPG/PNG | 1MB |
| IG Story | 1080×1920 | MP4 | 30s |
| Display Banner | 300×250, 728×90, 160×600 | PNG | 150KB |

**3. Visual Hierarchy**
- Primary element (largest, most prominent): product / person / headline
- Secondary element: supporting image or icon
- Tertiary: body copy, CTA button

**4. Copy on Creative**
- Headline (max 25 characters for FB): specify exact text
- Body copy (max 1 line on visual): specify exact text
- CTA button text (max 15 characters): specify exact text
- Legal/disclaimer text if required

**5. Brand Constraints**
- Approved color palette (hex codes)
- Logo placement: corner + minimum clear space
- Fonts permitted on creative
- Images: real people / product only / abstract allowed?

**6. Do Not Rules**
- List explicit restrictions (competitor mentions, claims not substantiated, etc.)

## Output Expectations
- Complete brief document for 3 ad variants (A/B/C)
- Moodboard references (3-5 reference image URLs or descriptions)

## Quality Criteria
- Brief completable by designer in one session without questions
- All dimensions and file specs match current platform requirements
- Hungarian text if target market is HU (account for longer word lengths in layout)
