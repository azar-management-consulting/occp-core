## Copy Standards

**Language register decision (set at brief start, never mix):**
- B2B / executive / formal service: `Ön/Önök` (formal)
- B2C / coaching / younger audience / e-commerce: `te/ti` (informal)
- Never mix within the same page

## Hungarian-Specific Rules
- Hungarian word order: topic first, then comment — mirror natural spoken rhythm
- Avoid anglicisms when a clear Hungarian equivalent exists
- Long compound nouns: break into phrases for readability (e.g., "vállalkozásfejlesztési megoldás" → "megoldás a vállalkozás fejlesztéséhez")
- Numbers: Hungarian format — `1 500 000 Ft` (space as thousand separator, not comma)
- Dates: `2026. március 26.` format
- GDPR notice must appear on all lead capture forms: "Az adatokat az Adatkezelési tájékoztató szerint kezeljük."

## Persuasion Structure (for sales/landing copy)
1. **Nyitó mondat:** azonosítás — "Ha Te is..." / "Tudod azt az érzést, amikor..."
2. **Fájdalom amplifikáció:** 2-3 bullet, jelen idő, olvasó-centrikus
3. **Megoldás pozicionálás:** a termék/szolgáltatás mint az egyetlen logikus következő lépés
4. **Bizonyíték:** konkrét szám, ügyfélnév (ha engedélyezett), eredmény
5. **CTA:** cselekvési ige + azonnali haszon ("Töltsd le most ingyen" / "Kérd az ajánlatot")

## Premium Copy Signals
- Specificity over vagueness: "37 vállalkozás" not "sok cég"
- Results over features: "3 hét alatt visszakapod a befektetést" not "gyors megtérülés"
- Authority markers: tapasztalat, referencia, módszertan neve
- Scarcity/urgency (only if real): "Az ajánlat március 31-ig érvényes"

## Output Expectations
- Full copy for requested format (landing section / email / ad / article)
- 2 headline variations (A/B)
- CTA text: 3 variations
- Character count per section if ad format

## Quality Criteria
- Zero anglicisms without deliberate brand reason
- Register consistent throughout (formal or informal, never mixed)
- Persuasion structure complete: problem → solution → proof → CTA
