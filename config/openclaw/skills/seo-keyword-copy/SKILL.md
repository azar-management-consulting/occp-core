## SEO Copy Standards

**Target:** Google HU SERP top-3 ranking, E-E-A-T compliance, schema.org 26.0 compatible

## Pre-Writing Requirements
- Primary keyword: 1 (provided or researched)
- Secondary keywords: 3-5 (LSI + related intent)
- Search intent classification: informational / navigational / commercial / transactional
- Competitor top-3 content analysis: note average word count, heading structure, featured snippet format

## Content Structure (Google-preferred)
```
H1: Primary keyword, naturally phrased, max 60 characters
  H2: Subtopic 1 (covers main search intent directly)
    H3: Supporting detail
  H2: Subtopic 2 (secondary intent or related question)
  H2: FAQ Section (targeting PAA/featured snippets)
    H3: Question 1 (exact match to common search query)
    H3: Question 2
  H2: Conclusion + CTA
```

## Keyword Density Rules
- Primary keyword: 1% density (1× per 100 words), first 100 words mandatory
- Natural integration — never force keyword into grammatically awkward position
- Hungarian morphology: count inflected forms as keyword occurrence (e.g., "tanfolyam", "tanfolyamon", "tanfolyamot" are all valid)

## E-E-A-T Signals
- Author byline with credentials (if article format)
- Publication date + last updated date
- Cite 2+ external authoritative sources (`.gov.hu`, `.edu`, industry bodies)
- Include first-hand experience or case data (Experience signal)

## Meta Tags
- Title tag: primary keyword first, 50-60 characters, brand name last
- Meta description: includes keyword, benefit statement, CTA, 140-155 characters
- Schema suggestion: `Article`, `FAQPage`, or `HowTo` — specify which applies

## Output Expectations
- Full article/page copy with H1/H2/H3 structure
- Meta title + meta description
- 5-question FAQ section
- Schema type recommendation

## Quality Criteria
- Word count matches competitor average ± 20%
- Featured snippet candidate: at least 1 H2/H3 with a direct 40-60 word answer below it
- No keyword stuffing: Yoast readability score green
