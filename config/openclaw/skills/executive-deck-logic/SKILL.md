## Executive Deck Principles

**Primary rule:** every slide earns the right to the next slide. No filler, no "agenda" slides, no "thank you" slides.

**Audience:** C-suite, board, senior partners — time-constrained, pattern-matching decision-makers

## Narrative Arc Selection

**Problem-Solution (for pitches, sales decks):**
World as it is → Why it's painful → World as it could be → Our path there → Ask

**Strategy-Decision (for board/management decks):**
Current state → Options analyzed → Recommended path → Resource ask → Timeline

**Performance-Review (for operations decks):**
Target → Actual → Gap analysis → Root cause → Actions → Next period forecast

## Slide Architecture Rules

| Slide Element | Rule |
|---------------|------|
| Slide headline | Complete sentence stating the conclusion (not a topic label) |
| Body content | 1 idea per slide. If tempted to add more — make 2 slides |
| Data charts | Title = the insight, not "Revenue Chart" |
| Bullet points | Max 4 per slide, max 8 words per bullet |
| Slide count | 10-15 for 30-min meeting; 5-7 for 10-min briefing |

## Headline Writing Rules
- Bad: "Sales Performance Q1"
- Good: "Q1 sales missed target by 12% due to 3 identifiable causes"
- Bad: "Market Opportunity"
- Good: "HU SME SaaS market is €240M with no dominant player"

## Data Visualization Hierarchy
1. **Single large number** — biggest impact, use for key metric
2. **Bar chart** — comparison across categories
3. **Line chart** — trend over time
4. **Table** — when exact values matter more than pattern
5. **Pie chart** — avoid unless showing 2-3 segments (never >5)

## Decision Slide Requirements
Every deck must end with a decision slide:
- Recommendation: clear, one option (not "we could do A or B")
- Ask: specific (funding amount, approval, resources, deadline)
- If not approved: explicit consequence stated
- Next step: specific action with owner and date

## Output Expectations
- Slide-by-slide outline with headline (complete sentence) per slide
- Content notes per slide (what data/chart/visual)
- Speaker notes for top 3 most important slides
- Decision slide fully written

## Quality Criteria
- Every slide headline is a complete declarative sentence
- No slide without a clear purpose in the narrative
- Deck holds 10-second test: each slide communicates its message in 10 seconds
