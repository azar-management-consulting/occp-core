## Design Aesthetic

**Target:** High-trust, enterprise-grade visual identity for B2B consulting, SaaS, or executive services

## Monochrome Palette Specification
- **Backgrounds:** pure white (#FFFFFF), off-white (#F8F8F6), light gray (#F0F0EE)
- **Text:** near-black (#1A1A1A) for body, true black (#000000) for headings only
- **Grays:** 4-step scale: #E0E0DE / #A0A09E / #606060 / #303030
- **Accent (single):** one high-contrast accent permitted — deep navy (#0A1628), forest (#1A3A2A), or burgundy (#2A0A14)
- **Forbidden:** gradients, multiple accent colors, drop shadows (use borders instead)

## Typography Rules
- H1: uppercase or small-caps, tracking +0.05em, weight 700
- H2: weight 600, no tracking adjustment
- Body: weight 400, line-height 1.6, max-width 68ch
- Caption: weight 400, size -2 steps from body, color #606060
- Recommended pairing: Inter (UI) + Playfair Display or DM Serif (editorial headings)

## Layout Principles
- Grid: 12-column, 24px gutters, 80px horizontal margin (desktop)
- Section separators: thin 1px borders (#E0E0DE) preferred over background color blocks
- White space: minimum 80px between major sections
- Photography: black-and-white or desaturated preferred; if color — muted, no vibrant saturated images
- Icons: line-only (stroke), 1.5px, never filled

## Application Contexts
- Executive pitch deck: dark cover slide (near-black BG, white text), white content slides
- Case study PDF: heavy whitespace, pull-quotes in large type
- Proposal document: clear information hierarchy, no decorative elements
- Website header: sticky, white BG, logo left, navigation right, no mega-menu

## Output Expectations
- Color palette spec with hex codes
- Typography pairing recommendation with scale
- Layout grid spec
- Sample slide/page description (hero + body layout)

## Quality Criteria
- Color contrast: all text/BG pairs AA compliant (minimum 4.5:1)
- No more than 3 font weights used in any single document
- Visual system conveys authority and precision at first glance
