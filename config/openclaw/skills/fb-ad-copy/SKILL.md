## Ad Copy Structure (per variation)

**Required elements:**
1. **Hook (first line):** must stop the scroll — question, bold statement, or relatable pain
2. **Body (2-4 sentences):** problem → solution → proof → bridge to CTA
3. **CTA sentence:** sentence ending with the action
4. **Headline (below creative):** max 40 characters, benefit-focused
5. **Description (optional):** max 30 characters, supporting detail or urgency

## Hungarian Meta Ad Rules
- First 125 characters are visible without "Tovább" — make them count
- Formal `Ön` for financial/legal/medical/corporate; informal `te` for lifestyle/e-commerce/coaching
- Price in copy: always include "Ft" suffix, use space as thousand separator
- Strong performs: "Miért..." questions, specific numbers, "Ingyen" (free)
- Weak performs: generic claims, "A legjobb", "Minőségi"

## Hook Formulas (proven for HU market)
- Question hook: "Miért nem működik a [common approach] 2026-ban?"
- Empathy hook: "Ha már megpróbáltad X-et és mégsem jött össze..."
- Curiosity hook: "Ez az 1 dolog, amit [audience] 90%-a rosszul csinál"
- Result hook: "[Specific result] — [timeframe]-ban. Így csináltuk."
- Mistake hook: "A legnagyobb hiba, amit [audience] elkövet"

## Deliverable: 3 Variations (A/B/C)
Each variation targets a different emotion:
- A: Pain/problem awareness (TOFU — cold audience)
- B: Solution/benefit (MOFU — warm, retargeting)
- C: Urgency/scarcity (BOFU — hottest audience, conversion)

## Output Format Per Variation
```
[HOOK]
[BODY — 2-4 sentences]
[CTA SENTENCE]
---
Headline: [max 40 chars]
Description: [max 30 chars]
CTA Button: [Tudj meg többet / Vásárlás / Feliratkozás / Ajánlatkérés]
Target: [Cold/Warm/Hot] | Placement: [Feed/Story/Reels]
```

## Quality Criteria
- Hook passes the "thumb-stop" test: would a scrolling user pause?
- No prohibited claims (health guarantees, income guarantees without disclaimer)
- Copy length: TOFU 150-250 chars body, BOFU 80-150 chars body (urgency is tighter)
