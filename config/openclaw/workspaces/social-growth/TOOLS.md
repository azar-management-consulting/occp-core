# social-growth — Allowed Tools

## Engedelyezett eszkozok
| Tool | Hasznalat | Korlat |
|------|-----------|--------|
| read | Kampany briefek, analytics adatok, brand guide | Korlatlan |
| write | Ad copy, kampany terv, script | Csak workspace-en belul |
| browser | Platform docs, trend kutatas, versenytars elemzes | Csak olvasas |

## TILTOTT eszkozok
- bash: NEM engedelyezett
- exec: NEM engedelyezett
- edit: NEM engedelyezett

## Output formatum
- Facebook Ad: primary text + headline + description + CTA button tipus
- Instagram Carousel: slide-onkenti szoveg + vizualis brief
- TikTok Script: idokod (0:00-0:03 hook, 0:03-0:15 content, 0:15-0:30 CTA)
- Kampany terv: cel → celcsoport → uzenet → kreativ → koltsegvetes → KPI

## Browser hasznalat
- Facebook Ad Library — versenytars hirdetes elemzes
- Instagram Explore — trend kutatas
- TikTok Creative Center — trending hangok, effektek
- Google Analytics / Meta Business Suite referenciak

## Sub-agent tool oroklodes
Minden sub-agent: read, write, browser (azonos korlatokkal)
