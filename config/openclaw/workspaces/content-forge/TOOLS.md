# content-forge — Allowed Tools

## Engedelyezett eszkozok
| Tool | Hasznalat | Korlat |
|------|-----------|--------|
| read | Meglevo tartalmak, brand guide, brief | Korlatlan |
| write | Szovegek, cikkek, email template-ek | Csak workspace-en belul |
| browser | Kulcsszo kutatas, versenytars tartalom, trendek | Csak olvasas |

## TILTOTT eszkozok
- bash: NEM engedelyezett
- exec: NEM engedelyezett
- edit: NEM engedelyezett

## Output formatum
- SEO cikk: Markdown, H1-H3 strukturaval, meta title + description
- Sales copy: headline + subheadline + body + CTA blokk
- Email: subject line + preheader + body + CTA + footer
- Esettanulmany: problema → megoldas → eredmeny (szamokkal)

## Browser hasznalat
- Google Trends, Ubersuggest, AnswerThePublic — kulcsszokutatas
- Versenytars oldalak tartalom elemzese
- Iparagi hirek es trendek
- Schema.org markup referenciak

## Minosegi checklist (kotelezoe minden output-hoz)
- [ ] Helyesiras ellenorizve
- [ ] Kulcsszavak termeszetesen beepitve
- [ ] CTA egyertelmu es cselekvesorientalt
- [ ] Hangnem konzisztens a brand-del
- [ ] Nincs plagiarizmus

## Sub-agent tool oroklodes
Minden sub-agent: read, write, browser (azonos korlatokkal)
