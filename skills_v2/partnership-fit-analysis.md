## Partnership Evaluation Framework

**Partnership types to distinguish:**
- **Referral:** revenue share on leads/deals
- **Integration:** technical product integration (API/plugin)
- **Co-marketing:** shared campaigns, joint content
- **Reseller:** partner sells our product under their brand or as bundle
- **Strategic alliance:** shared roadmap, co-development, market co-entry

## Fit Scoring Matrix

Score each dimension 1-5 (1=poor fit, 5=excellent fit):

| Dimension | Weight | Score | Weighted |
|-----------|--------|-------|---------|
| ICP overlap (same target customer) | 25% | | |
| Complementary (not competing) products | 20% | | |
| Market reach (partner audience size/quality) | 15% | | |
| Reputation and brand alignment | 15% | | |
| Technical integration feasibility | 10% | | |
| Partner motivation (what they gain) | 10% | | |
| Legal and compliance compatibility | 5% | | |
| **Total** | **100%** | | |

**Threshold:** ≥ 3.5 weighted average → pursue. < 2.5 → decline.

## Synergy Map

Identify specific synergies per partnership type:
- **Revenue synergy:** joint upsell potential, referral volume estimate
- **Product synergy:** features unlocked by integration (specific user benefit)
- **Market synergy:** new segment or geography accessible through partner
- **Credibility synergy:** brand halo effect from partner association

## Hungarian Partnership Context
- Formalize with written agreement before any joint marketing (HU business culture: verbal agreement is weak)
- Reference: Hungarian Civil Code (Ptk.) governs commercial agreements
- Non-compete clause: specify clearly (Hungarian courts enforce if reasonable in scope/duration)
- Revenue split standard in HU: 15-25% referral fee for leads, 25-40% for closed deals

## Deal Structure Template
```
Partnership Type: [Referral / Integration / Co-marketing / Reseller / Strategic]
Revenue Share: X% on [closed deals / MRR / one-time]
Exclusivity: None / Category-exclusive / Territory-exclusive
Term: [12 months] with [3-month] renewal notice
Termination: [30-day] notice, no penalty after [6-month] minimum
KPIs: [leads/month, integration installs, co-marketing events/quarter]
Review cadence: [quarterly business review]
```

## Output Expectations
- Completed scoring matrix for each candidate partner
- Synergy map (3-5 specific synergies)
- Deal structure recommendation
- Go/No-Go with 2-sentence rationale

## Quality Criteria
- Score based on verifiable data (company website, LinkedIn, product review)
- Partner motivation explicitly analyzed — partnerships fail when only one party gains
- Deal structure protects against exclusivity lock-in unless strategic value is clear
