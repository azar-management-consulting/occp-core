## Script Structure

**Total length:** 30-60 seconds optimal (algorithm promotes completion rate)
**Format output:** timestamped script with visual/action notes

## The 3-Second Hook Rule
First 3 seconds determine if viewer stays. Hook options:
- **Visual hook:** start mid-action, unusual visual, text overlay with bold claim
- **Verbal hook (spoken):** "Ezt senki nem mondja el neked..." / "Stop — ha [audience], nézd ezt"
- **Text hook (overlay):** bold claim on screen before speaking begins
- **Pattern interrupt:** unexpected opening that defies viewer expectation

## Script Template
```
[0:00-0:03] HOOK
  Spoken: "..."
  Visual: [describe what's on screen]
  Text overlay: "..." (optional)

[0:03-0:15] SETUP / PROBLEM
  Spoken: "..."
  Visual: [talking head / b-roll / text]

[0:15-0:45] CONTENT / VALUE DELIVERY
  Point 1 [0:15-0:25]: ...
  Point 2 [0:25-0:35]: ...
  Point 3 [0:35-0:45]: ...

[0:45-0:55] PAYOFF / RESOLUTION

[0:55-1:00] CTA
  Spoken: "Kövesd, mert holnap folytatom" / "Mentsd el"
  Text overlay: "@handle" or "Rész 2 jön"
```

## Engagement Mechanics
- Open loop: tease the answer at start, deliver at end
- Pattern interrupt every 7-10 seconds (cut, zoom, text pop, sound effect)
- "Part 2" mechanic: end on a cliff-hanger for series content
- Comment bait: ask a yes/no question viewers can answer in comments
- Save bait: "Mentsd el ezt, mert szükséged lesz rá"

## Hungarian TikTok/Reels Notes
- Informal `te` mandatory — formal `Ön` performs poorly on short video
- Trending audio boosts reach — suggest 1 trending audio direction if known
- Hungarian subtitles increase watch time (+30% retention average)
- Niche hashtags (#magyarmarketing, #magyarivállalkozó) outperform global ones

## Output Expectations
- Full timestamped script (spoken + visual notes)
- 3 hook variations for A/B testing
- Hashtag set (8-10 tags: 3 niche, 3 mid, 2 broad)
- Suggested caption (first line = hook repeat)

## Quality Criteria
- Script readable aloud in target time (test: read at natural pace)
- Every 10-second block adds value independently
- CTA specific and actionable (not just "like and subscribe")
